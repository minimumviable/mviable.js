Create

Minimum Viable works with any HTML5 site or application. Just include our open source JavaScript API in your page (we recommend putting it in the <head> element, but do what works for you).

Building from scratch? HTML5 Boilerplate is a great way to get started.

Login

Protect your users' data with an OAuth2 login such as Google or Facebook. To redirect your users to the OAuth2 login page just call login, passing the provider name as the parameter.

Sync

Syncronize data in HTML5 localStorage with Minimum Viable's cloud storage using the sync function. To be notified when the syncronization is complete, you can register an event listener.

Deploy

When you're ready to deploy, add your application's domain names to your Minimum Viable account. Any request coming from that origin will automatically be routed to your application.
